# COVID-19-Arabic-Emotion-Dataset
<b>AraEmoCorpus:</b>
The corpus contains more than 5.5 million Arabic tweets tagged with 6 emotion categories:
1.	Anger
2.	Disgust
3.	Fear
4.	Joy
5.	Sadness
6.	Surprise
